# Examples Reference

This folder contains reference outputs for the early analysis and strategy phases of the kit.

## What belongs here

- OpenAPI review examples
- Auth and limits analysis examples
- Pagination, filtering, and sorting review examples
- Testing pattern examples
- OAS snapshot examples
- Comprehensive strategy examples

## What does not belong here

- Runnable collections that should be copied into a project
- Real execution outputs or raw runner artifacts
- Any environment file containing secrets or current values

Use `templates/api-pack/` when you need the canonical runnable starter pack.
